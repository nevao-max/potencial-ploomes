para a sua operação industrial

O Campo Dev cria botões, cálculos, indicadores visuais e lógicas de negócio específicas da indústria diretamente no Ploomes — sem desenvolvimento externo.

Exemplo 1 · Botão — Abrir Pedido no ERP
Botão que abre o pedido correspondente no ERP externo diretamente do card do CRM — sem sair da plataforma.
// Abre o pedido no ERP ao clicar if (card.erp_order_id) { return { label: "📦 Abrir no ERP", url: `https://erp.empresa.com/pedido/ ${card.erp_order_id}`, style: "primary" }; }
Exemplo 2 · Semáforo de Saúde do Distribuidor
Calcula automaticamente a saúde do distribuidor com base em volume, frequência e mix — exibindo indicador visual no card.
// Score de saúde do canal let score = 0; if (card.monthly_vol > 50000) score += 40; if (card.days_since_order < 15) score += 35; if (card.active_products > 5) score += 25; return { value: score + "%", icon: score > 70 ? "🟢" : score > 40 ? "🟡" : "🔴" };
Exemplo 3 · Alerta de Contrato Vencendo
Exibe aviso visual no card quando o contrato de fornecimento está dentro do prazo de renovação configurado.
// Alerta de vencimento const dias = Math.floor( (new Date(card.contract_end) - new Date()) / 86400000 ); if (dias <= 60) { return { label: `⚠️ Vence em ${dias} dias`, color: dias <= 30 ? "red":"orange" }; }
